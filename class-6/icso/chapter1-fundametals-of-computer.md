# Chapter 1: Fundamentals of Computer

## Grade 6 ICSO Study Notes

## 1. What is a Computer?

A **computer** is an electronic machine that accepts data, processes it, stores it, and gives useful results.

### Simple Example

When you type `5 + 3` on a calculator:

| Step       | What happens                       |
| ---------- | ---------------------------------- |
| Input      | You press 5, +, 3                  |
| Processing | The computer calculates the answer |
| Output     | It shows 8                         |
| Storage    | It may save the result if needed   |

This is called the **IPO cycle**:

**Input → Processing → Output**

Sometimes we also include storage:

**Input → Processing → Output → Storage**

---

# 2. Data and Information

## Data

**Data** means raw facts.

Examples:

* 90
* Ruthvik
* 25°C
* 8, 6, 9, 7

## Information

**Information** is processed data that has meaning.

Example:

Raw data: `8, 6, 9, 7`
Information: “Ruthvik scored an average of 7.5 marks.”

---

# 3. Main Parts of a Computer System

A computer system has two major parts:

1. **Hardware**
2. **Software**

---

# 4. Hardware

## What is Hardware?

**Hardware** means the physical parts of a computer that we can see and touch.

Examples:

* Monitor
* Keyboard
* Mouse
* CPU
* Printer
* Scanner
* Motherboard
* RAM
* Hard disk
* Speaker

## Simple Rule

If you can touch it, it is usually **hardware**.

---

# 5. Software

## What is Software?

**Software** is a set of instructions that tells the computer hardware what to do.

Examples:

* Windows
* Android
* MS Word
* Google Chrome
* Paint
* Antivirus
* Calculator app

## Simple Rule

You cannot physically touch software.
It runs on hardware.

---

# 6. Difference Between Hardware and Software

| Hardware                     | Software                           |
| ---------------------------- | ---------------------------------- |
| Physical parts of a computer | Programs/instructions              |
| Can be touched               | Cannot be touched                  |
| Works with help of software  | Runs on hardware                   |
| Examples: CPU, RAM, printer  | Examples: Windows, MS Word, Chrome |

Example:

A **printer** is hardware.
The **printer driver** is software that helps the computer communicate with the printer.

---

# 7. Types of Software

Software is mainly divided into:

1. **System Software**
2. **Application Software**

---

## 7.1 System Software

**System software** helps the computer run and manage its hardware.

It acts like a manager between the user, hardware, and application software.

### Examples of System Software

* Operating system
* Device drivers
* Utility software
* Language processors
* Firmware

---

## 7.2 Operating System

An **Operating System**, or **OS**, is the most important system software.

It manages all computer resources and provides an interface between the user and the computer.

### Examples

* Windows
* Linux
* macOS
* Android
* iOS
* Ubuntu

### Functions of Operating System

The OS:

* Manages files and folders
* Manages memory
* Manages input and output devices
* Runs application software
* Provides user interface
* Controls hardware resources

### Important Point

Android is an operating system, not an application software.

---

## 7.3 Application Software

**Application software** is used to perform specific tasks for the user.

### Examples

| Task                 | Application           |
| -------------------- | --------------------- |
| Typing documents     | MS Word               |
| Making presentations | PowerPoint            |
| Browsing internet    | Chrome, Edge, Firefox |
| Drawing              | Paint                 |
| Editing images       | Photoshop             |
| Playing videos       | VLC Media Player      |

### Types of Application Software

#### General-purpose software

Used by many people for common tasks.

Examples:

* MS Word
* Excel
* PowerPoint
* Chrome

#### Specific-purpose software

Made for a specific job.

Examples:

* Railway reservation software
* School fee management software
* Hospital management software
* Banking software

---

# 8. Utility Software

**Utility software** helps maintain, protect, and improve the computer system.

### Examples

| Utility Software     | Use                             |
| -------------------- | ------------------------------- |
| Antivirus            | Protects from viruses           |
| Disk Cleanup         | Removes unnecessary files       |
| WinZip               | Compresses files                |
| Backup software      | Saves copies of important files |
| File management tool | Helps manage files              |

### Important Point

Utility software is a type of **system software**.

---

# 9. Device Driver

A **device driver** is system software that helps the operating system communicate with a specific hardware device.

### Examples

* Printer driver
* Mouse driver
* Keyboard driver
* Graphics driver
* Sound driver

### Simple Example

If you connect a printer to your computer, the computer needs a **printer driver** to understand how to send printing instructions to that printer.

---

# 10. Firmware

**Firmware** is a special type of software stored permanently inside hardware.

It helps hardware start and work properly.

### Example

**BIOS** is firmware.

BIOS helps start the computer when it is switched on.

---

# 11. BIOS and POST

## BIOS

**BIOS** stands for **Basic Input Output System**.

It is stored in non-volatile memory.

BIOS helps initialize hardware during the booting process.

## POST

**POST** stands for **Power-On Self-Test**.

It checks whether important hardware parts are working properly before the operating system loads.

### Simple Sequence

1. You switch on the computer.
2. BIOS starts.
3. POST checks hardware.
4. Operating system loads.

---

# 12. Language Processors

Computers understand machine language, which is made of 0s and 1s.

Humans write programs in languages like Python, Java, or C++.
Language processors convert human-written code into machine language.

### Types

| Language Processor | Use                                               |
| ------------------ | ------------------------------------------------- |
| Assembler          | Converts assembly language into machine code      |
| Compiler           | Converts entire program into machine code at once |
| Interpreter        | Converts and runs program line by line            |

---

# 13. Debugger

A **debugger** is a program used to find and fix errors in other programs.

An error in a program is called a **bug**.

Example:

If a game crashes because of a coding mistake, developers may use a debugger to find the problem.

---

# 14. Internal and External Hardware

## Internal Hardware

Internal hardware parts are located inside the system unit.

Examples:

* Motherboard
* CPU
* RAM
* Hard disk
* GPU
* Expansion cards

## External Hardware

External hardware parts are connected from outside.

Examples:

* Keyboard
* Mouse
* Monitor
* Printer
* Scanner
* Speaker
* Webcam
* USB drive
* External hard drive

---

# 15. Motherboard

The **motherboard** is the main circuit board of a computer.

It connects all important parts of the computer.

### Functions of Motherboard

* Connects CPU, RAM, storage, and other parts
* Allows communication between parts
* Has slots for expansion cards
* Has ports for connecting devices

### Important Point

The motherboard does **not** contain instructions telling all hardware what to do.
That is the job of software.

---

# 16. CPU

## What is CPU?

CPU stands for **Central Processing Unit**.

It is called the **brain of the computer** because it processes instructions and data.

### Main Parts of CPU

1. **Control Unit**
2. **ALU**
3. **Memory Unit / Registers**

---

## 16.1 Control Unit

The **Control Unit** controls the flow of data and instructions inside the CPU.

It is responsible for:

* Fetching instructions
* Decoding instructions
* Controlling data flow
* Coordinating different parts of the computer

---

## 16.2 ALU

ALU stands for **Arithmetic Logic Unit**.

It performs:

### Arithmetic operations

* Addition
* Subtraction
* Multiplication
* Division

### Logical operations

* Greater than
* Less than
* Equal to
* True/False decisions

---

## 16.3 Memory Unit

The memory unit temporarily stores data and instructions needed by the CPU.

---

# 17. RAM

RAM stands for **Random Access Memory**.

It is the temporary memory of a computer.

### Features of RAM

* Stores data temporarily
* Very fast
* Used while programs are running
* Data is lost when power is switched off

RAM is called **volatile memory**.

### Example

When you open a game, it loads into RAM so that it can run quickly.

---

# 18. ROM

ROM stands for **Read Only Memory**.

It stores important instructions permanently.

### Features of ROM

* Permanent memory
* Data is not lost when power is switched off
* Stores startup instructions

ROM is called **non-volatile memory**.

---

# 19. Storage Devices

Storage devices save data for future use.

## Volatile Storage

Data is lost when power goes off.

Example:

* RAM

## Non-volatile Storage

Data remains saved even after power is switched off.

Examples:

* Hard disk
* SSD
* USB drive
* Memory card
* CD/DVD

---

# 20. Hard Disk

A **hard disk** is a storage device used to store data permanently.

It is a **non-volatile storage device**.

Examples of data stored on hard disk:

* Photos
* Videos
* Documents
* Games
* Operating system

---

# 21. USB Drive

A **USB drive**, also called a pen drive or flash drive, is a portable storage device.

It is external hardware and non-volatile storage.

---

# 22. Expansion Slot

An **expansion slot** is a socket on the motherboard where extra hardware cards can be added.

Examples of expansion cards:

* Graphics card
* Sound card
* Network card

---

# 23. GPU

GPU stands for **Graphics Processing Unit**.

It is used to process images, videos, animations, and graphics.

### Uses of GPU

* Gaming
* Video editing
* 3D graphics
* Artificial intelligence
* Machine learning
* Image processing

### Difference Between CPU and GPU

| CPU                         | GPU                                                  |
| --------------------------- | ---------------------------------------------------- |
| General-purpose processor   | Specially good at graphics and parallel calculations |
| Handles many types of tasks | Handles image/video/AI calculations faster           |
| Brain of computer           | Graphics/math specialist                             |

---

# 24. NIC

NIC stands for **Network Interface Card**.

It helps a computer connect to a network.

### Use

NIC allows computers to communicate with each other over a network.

Example:

When a computer connects to the internet using Ethernet or Wi-Fi, a network card helps with communication.

---

### Examples

* Keyboard
* Mouse

* Trackball

* Barcode reader

# 26. Output Devices

# 27. Input and Output Device Difference

| Input Device                       | Output Device                       |
| ---------------------------------- | ----------------------------------- |
| Sends data to computer             | Gives result from computer          |
| User → Computer                    | Computer → User                     |
| Examples: keyboard, mouse, scanner | Examples: monitor, printer, speaker |

---

# 28. Devices That Can Be Both Input and Output

Some devices can both send data to the computer and receive data from it.

Examples:

* Touchscreen
* Headset with microphone
* Modem
* Network card

---

# 29. Mouse

A mouse is an input device used to point, click, drag, and select items.

---

# 30. Trackball

A **trackball** is a pointing device with a ball on top.

The user rotates the ball to move the pointer.

### Features

* Used in video games and flight simulators
* Can be wired or wireless
* Helpful where there is less desk space

---

# 31. Joystick

A **joystick** is an input device mostly used for games and simulators.

Examples:

* Flight simulator
* Racing games
* Video games

---

# 32. Light Pen

A **light pen** is an input device used to point or draw directly on the screen.

---

# 33. Digitizer

A **digitizer** is an input device used to convert drawings or handwritten work into digital form.

It often looks like a tablet with a special pen.

### Example

Artists use a digitizer tablet to draw on a computer.

---

# 34. Scanner

A **scanner** is an input device that converts paper documents or pictures into digital form.

### Example

If you scan a drawing, the computer stores it as an image file.

---

# 35. OCR

OCR stands for **Optical Character Recognition**.

It reads printed or handwritten text and converts it into digital text.

### Example

You scan a page from a book.
OCR converts the image of text into editable computer text.

---

# 36. OMR

OMR stands for **Optical Mark Recognition**.

It reads marks made in specific places.

### Example

OMR is used to check multiple-choice answer sheets where students darken bubbles.

---

# 37. MICR

MICR stands for **Magnetic Ink Character Recognition**.

It reads special characters printed with magnetic ink.

### Example

MICR is used in banks to read cheque numbers.

---

# 38. Barcode Reader

A **barcode reader** scans black and white lines and converts them into digital data.

### Uses

* Shops for billing
* Warehouses
* Courier and logistics tracking
* Libraries
* Product identification

### Example

At a supermarket, the cashier scans a product barcode to get its price.

---

# 39. Printer

A **printer** is an output device that prints text and images on paper.

Printed output is called **hard copy**.

---

# 40. Types of Printers

Printers are mainly divided into:

1. **Impact Printers**
2. **Non-impact Printers**

---

## 40.1 Impact Printers

Impact printers print by striking an ink ribbon against paper.

### Example

* Dot-matrix printer
* Daisy-wheel printer

They are usually noisy.

---

## 40.2 Non-impact Printers

Non-impact printers do not strike the paper.

They are usually quieter and faster.

### Examples

* Inkjet printer
* Laser printer
* Thermal printer

---

# 41. Inkjet Printer

An **inkjet printer** sprays tiny drops of ink onto paper.

### Features

* Good for colour printing
* Uses spray technology
* Common in homes and small offices

---

# 42. Laser Printer

A **laser printer** uses laser technology and toner powder.

### Features

* Fast
* High-quality printing
* Common in offices

---

# 43. Thermal Printer

A **thermal printer** uses heat to print.

### Uses

* Receipts
* ATM slips
* Billing counters

---

# 44. Dot-Matrix Printer

A **dot-matrix printer** is an impact printer.

It prints using pins that strike an ink ribbon.

---

# 45. Plotter

A **plotter** is an output device used to print large drawings.

### Uses

* Engineering drawings
* Maps
* Architectural plans
* Large technical designs

### Important Point

A plotter draws continuous lines using pens.

---

# 46. Projector

A **projector** is an output device that displays images or videos on a large screen or wall.

### Uses

* Classrooms
* Presentations
* Movie halls
* Meetings

---

# 47. Speaker

A **speaker** is an output device.

It converts digital audio signals into sound waves that we can hear.

### Types

* Wired speaker
* Wireless speaker
* Bluetooth speaker

---

# 48. Headphones

Headphones are output devices used to listen to sound privately.

If headphones have a microphone, the full device can work as both input and output.

---

# 49. Microphone

A **microphone** is an input device.

It records voice or sound and sends it to the computer.

---

# 50. Webcam

A **webcam** is an input device that captures photos and videos.

### Uses

* Video calls
* Online classes
* Recording videos

---

# 51. Monitor

A **monitor** is an output device.

It displays text, images, videos, and other results.

Output shown on a monitor is called **soft copy**.

---

# 53. Microcomputer

Everyday computers used by people are usually microcomputers.

---

# 54. Desktop Computer

A desktop computer is kept on a desk.

It usually has:

* Monitor
* Keyboard
* Mouse
* System unit

---

# 55. Laptop

A laptop is a portable computer.

It has screen, keyboard, touchpad, battery, and speakers in one unit.

---

# 56. Tablet

A tablet is a portable computer with a touchscreen.

---

# 57. Smartphone

A smartphone is a small handheld computer that can make calls and run apps.

Smartphones and tablets can run many apps for:

* Productivity
* Entertainment
* Communication
* Learning

---

# 58. Minicomputer

A **minicomputer** is more powerful than a microcomputer but smaller than a mainframe.

It can support multiple users.

---

# 59. Mainframe Computer

A **mainframe computer** is a large and powerful computer used by big organizations.

### Uses

* Banks
* Airlines
* Government offices
* Large companies

### Features

* Handles huge amounts of data
* Supports many users
* Very reliable
* Has large memory and high processing power

---

# 60. Supercomputer

A **supercomputer** is one of the fastest and most powerful computers.

It performs calculations at extremely high speed.

### Uses

* Weather forecasting
* Scientific research
* Space research
* Climate modelling
* Nuclear research
* Complex simulations

### Important Point

Supercomputer performance is measured in **FLOPS**.

FLOPS means **Floating Point Operations Per Second**.

### Examples

* Fugaku
* Summit

---

# 61. Server

A **server** is a computer that provides services or resources to other computers over a network.

The computers that use server services are called **clients**.

### Example

When students access school files from many computers, those files may be stored on a server.

---

# 62. Analog Computer

An **analog computer** works with continuous data.

It measures physical quantities like:

* Temperature
* Speed
* Pressure
* Voltage

### Example

* Speedometer
* Some medical equipment

---

# 63. Digital Computer

A **digital computer** works with discrete data, usually 0s and 1s.

Most modern computers are digital computers.

### Examples

* Laptop
* Desktop
* Smartphone
* Calculator

---

# 64. Hybrid Computer

A **hybrid computer** has features of both analog and digital computers.

### Example

Medical equipment like ECG machines can be considered hybrid because they measure physical signals and process/display them digitally.

---

# 65. Special-purpose Computer

A **special-purpose computer** is designed to do a specific task.

### Examples

* ATM
* Gaming console
* Washing machine controller
* Calculator
* Traffic light control system

A laptop is not usually called a special-purpose computer because it can do many tasks.

---

# 66. General-purpose Computer

A **general-purpose computer** can perform many different tasks.

Examples:

* Laptop
* Desktop
* Tablet
* Smartphone

---

# 67. Ports and Cables

### Common Ports

Hardware can be upgraded by adding or replacing physical parts.

Examples:

* Add more RAM
* Replace hard disk with SSD
* Add graphics card
* Replace keyboard or monitor

## Software Upgrade

Software can be upgraded by installing a newer version.

Examples:

* Updating Windows
* Updating Chrome
* Installing latest antivirus version
* Updating Android apps

# 71. Exam Tips for Bui

Read the question carefully and ask:

1. Is it asking about **hardware or software**?
2. Is the device giving **input or output**?
3. Is the memory **temporary or permanent**?
4. Is the computer type based on **size, speed, purpose, or data type**?
5. Is the software **system software or application software**?
6. Is the device inside the system unit or connected externally?

Most mistakes happen because options look similar. Always eliminate wrong options first.
